This folder contains exercises and scripts focused on mastering **core Python syntax and language features**. This work is the essential foundation for moving into specialized Data Science libraries (like Pandas and NumPy).

## üéØ Key Skills Demonstrated

The files in this directory demonstrate proficiency in the following fundamental programming concepts:

* **Input/Output (I/O):** Correctly taking user input using `input()` and formatting output with `print()`.
* **String Manipulation:** Using built-in string methods (e.g., `.lower()`, `.strip()`, `.replace()`) and utilizing f-strings for efficient formatting.
* **Control Flow:** Implementing conditional logic (`if`, `elif`, `else`) and iteration (`for` and `while` loops).
* **Data Structures:** Basic usage of Python's built-in containers like **Lists**, **Dictionaries**, and **Tuples**.
* **Functions:** Defining and calling simple custom functions with clear parameters and return values, including the use of **Docstrings**.
